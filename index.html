Burn Protocol

Burn Protocol is an experimental on-chain mechanism (mecanismo experimental en blockchain) that allows users to burn ERC20 tokens (quemar tokens ERC20) in a verifiable and transparent way (forma verificable y transparente).

What is this project?

This project is a technical protocol (protocolo técnico) designed to reduce token supply (reducir la oferta de tokens) through voluntary burn actions (acciones voluntarias de quema).

What this project is NOT

  • Not an investment (no es una inversión)
  • Not a financial product (no es un producto financiero)
  • No profit or returns are promised (no promete ganancias)

How does it work?

Users approve a smart contract (autorizan un contrato inteligente) to transfer tokens to a dead address (dirección muerta), permanently removing them from circulation (eliminándolos permanentemente de circulación).
